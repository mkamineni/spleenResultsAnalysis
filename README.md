# Spleen Results Analysis

## Description

Downstream statistical analyses, visualization, and figure generation for studies investigating associations between spleen imaging-derived phenotypes, genetics, and coronary artery disease (CAD).

## Overview

Emerging evidence suggests that the spleen plays an important role in inflammatory pathways and cardiovascular disease. This repository contains code used for downstream analyses of spleen imaging-derived phenotypes, including statistical modeling, visualization, genetic follow-up analyses, and generation of publication figures.

Analyses integrate spleen radiomic features with clinical phenotypes, cardiovascular outcomes, adiposity measures, blood biomarkers, and genetic data to identify biological pathways linking spleen imaging phenotypes to coronary artery disease.

## Repository Structure

- **Visualization scripts**: generation of forest plots, radiomics feature plots, heatmaps, and summary figures
- **Biomarker analyses**: associations between spleen imaging features, adiposity measures, and blood biomarkers
- **Genetic follow-up analyses**: overlap of spleen-associated variants with CAD loci, SNP comparison analyses, and gene prioritization
- **Gene prioritization**: integration of PoPS and nearest-gene approaches for candidate gene identification
